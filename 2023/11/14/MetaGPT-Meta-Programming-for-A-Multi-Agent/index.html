MetaGPT: Meta Programming for A Multi-Agent

MetaGPT: Meta Programming for A Multi-Agent Collaborative Framework
HongSirui(DeepWisdom)
arxiv
2023-8

研究问题（背景、目的及结论）

背景： 最近，通过使用大型语言模型( LLMs )驱动的多自主体，在自动化任务解决方面取得了显著的进展。然而，现有的工作主要集中在简单任务上，缺乏对复杂任务的探索和研究，主要原因是幻觉问题。当多个智能体相互作用时，这种幻觉会被无限放大，从而导致在处理复杂问题时失败。目的： 引入了MetaGPT,将SOP工作流程引入，同时为多代理分配了不同的角色，建立了一种解决复杂问题的框架。结论： 开发了metaGPT，同时，本项目突出了将人类领域知识纳入多智能体的潜力，从而为应对复杂的现实世界挑战开辟了新的途径。

总结（结果与讨论）

在这项工作中，提出了 MetaGPT，这是一个通过 SOP 进行协作代理的有前途的框架，它管理 LLM 以模仿高效的人类工作流程。为了将SOP编码为提示，MetaGPT通过角色定义、任务分解、流程标准化等技术设计来管理多Agent，最终完成只有一行需求的端到端开发流程。我们在软件开发中的说明性示例通过详细的 SOP 和提示展示了该框架的潜力。实验结果表明，我们的 MetaGPT 可以产生相对于现有会话和基于聊天的多代理系统具有更高一致性的综合解决方案。我们相信，这项工作为多智能体交互和合作的方式开辟了新的可能性，重新定义了复杂问题解决的前景，并指出了通向通用人工智能的潜在途径。
